# 🍔 Food Delivery Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Food Delivery Dashboard** built using **Power BI** to analyze business performance, customer behavior, and delivery efficiency. The dashboard transforms raw data into meaningful insights for better decision-making.

## 🎯 Objectives

* Analyze total orders and revenue trends
* Understand customer behavior and preferences
* Identify top-performing food categories and restaurants
* Evaluate delivery performance and efficiency
* Enable interactive data exploration

---

## 📊 Key Features

### 🔹 KPIs (Key Performance Indicators)

* Total Orders
* Total Revenue
* Average Order Value (AOV)
* Delivery Time
* Customer Count

---

### 🔹 Visualizations

* 📈 Orders & Revenue Trends (Daily/Monthly)
* 🍕 Category-wise Sales Analysis
* 🏪 Restaurant Performance
* 🌍 Location-based Order Distribution
* ⭐ Customer Ratings & Behavior

---

### 🔹 Filters (Slicers)

* Date Filter
* Food Category Filter
* Location Filter

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

## 🔍 Insights Derived

* Peak order times occur on weekends
* Certain food categories generate higher revenue
* Delivery time impacts customer satisfaction
* Specific locations show higher demand

---

## 🚀 How to Use

1. Open the dashboard using the link above

## 💼 Use Case

This dashboard can help:
